- Calcolo contratti totali, inseriti, mese elettrico corrente.

- Calcolo contratti totali inseriti, di tipo DOMESTICO nel mese elettrico corrente.

- Calcolo contratti totali inseriti, di tipo Altri Usi nel mese elettrico corrente.
